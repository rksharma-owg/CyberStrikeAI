# Screenshot Guidelines

This folder is reserved for curated documentation screenshots. Existing product
images remain in the repository's top-level `images/` directory; new documentation
flows may migrate here over time when their links are updated in the same pull
request.

Before adding a screenshot:

- use a local demonstration environment with synthetic data;
- remove tokens, credentials, hostnames, personal data, and real findings;
- crop unrelated desktop or browser content;
- provide descriptive alt text where the image is embedded;
- prefer PNG for interfaces and SVG for diagrams;
- use a stable, lowercase, hyphenated filename;
- update or remove the image when the documented interface changes.

Do not add screenshots solely for decoration. Each image should explain a workflow,
state, or decision that is harder to understand from text alone.
